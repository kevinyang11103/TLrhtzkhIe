# Python学生成绩管理系统 python728

## 项目概述

本项目是基于Python语言的学生成绩管理系统，采用前后端分离的架构设计，前端使用Vue.js框架，后端采用Django框架配合MySQL数据库。

## 技术栈

- Python
- Django
- Vue.js
- MySQL

## 系统角色

- 管理员
- 教师
- 学生

## 系统特性

### 前后端分离

系统采用前后端分离的设计，前端和后端可以独立部署与开发。

### 文档与模板

提供详细的项目文档及Visio图表模板，包括但不限于以下类型：

- 用例图
- ER图
- 结构图
- 实体属性图
- 时序图
- 流程图

### 开发工具与教程

- 提供项目所需全部软件。
- 提供视频调试教程。
- 提供同类技术开发项目系统核心技术讲解视频。

### 自定义与修改

- 提供图文教程，指导如何修改注册登录背景图、标题、项目名、数据库名、时间信息等。

## 运行环境

- 电脑配置要求：Node版本高于14（前端运行环境）。

## 部署步骤

（具体部署步骤将根据实际项目需求提供，以下为通用步骤）

1. 安装Python环境。
2. 安装MySQL数据库。
3. 配置Django后端环境。
4. 配置Vue.js前端环境。
5. 部署前后端项目。
